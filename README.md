# Triangle

Tells user what kind of triangle they have when given three dimensions.

## Setup Instructions

### Gems used
```erb
sinatra
sinatra-contrib
```
### Install Bundler
```erb
$ gem install bundler
```

### Run Bundler
```erb
$ bundle
```

### Bug Reports
Entering letters and special characters will cause program to run incorrectly
